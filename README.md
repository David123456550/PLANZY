Planzy es una aplicación diseñada para crear, gestionar y unirse a planes de cualquier tipo: desde una quedada para hacer senderismo hasta organizar un partido de pádel donde cada participante aporta dinero para pagar la pista. La app facilita la organización, comunicación y participación en actividades sociales de forma sencilla e intuitiva.

1. Inicio  
En esta sección se muestran todos los planes creados por los usuarios. Cada plan incluye:

- Ubicación  
- Precio (o indicación de que es gratuito)  
- Imagen representativa  
- Hora del evento  
- Chat del plan  
- Número de participantes  
- Información general del organizador  

El usuario puede explorar, filtrar y unirse a los planes disponibles.

2. Crear un plan  
Los usuarios pueden crear su propio plan personalizando:

- Si el plan es de pago o gratuito  
- Ubicación exacta  
- Hora de la quedada  
- Número máximo de asistentes  
- Imagen o descripción del plan  

Una vez creado, el plan aparecerá en el inicio para que otros usuarios puedan unirse.

3. Registro e inicio de sesión  
La app cuenta con un sistema de autenticación seguro:

- Registro con verificación por correo electrónico
- Inicio de sesión con credenciales  
- Recuperación de contraseña  

El correo de verificación es obligatorio para activar la cuenta y garantizar la seguridad de la comunidad.

4. Chats  
La aplicación incluye un sistema de mensajería dividido en dos tipos:

- Chats privados entre usuarios  
- Chats de grupo asociados a cada plan  

Esto permite coordinar detalles, resolver dudas y mantener una comunicación fluida antes y después del evento.


5. Tipos de planes (Suscripciones)  
Planzy ofrece diferentes modalidades para mejorar la experiencia:

Simple Funciones básicas: crear planes, unirse, chatear. |
Premium Ventajas adicionales como mayor visibilidad, creación ilimitada de planes, funciones avanzadas de organización, etc. 

6.Personalización: Modo oscuro / claro  
La app permite elegir entre:

- Modo claro  
- Modo oscuro  

El usuario puede cambiarlo en cualquier momento según su preferencia.


7.Idiomas disponibles  
Planzy está disponible en:

- Español  
- Inglés  

El idioma puede seleccionarse desde la configuración.
